Доступные функции

World
  • world.getEnemies() - Функция возвращает массив всех врагов в мире.
  • world.getBonuses() - Функция возвращает массив всех бонусов в мире.

Bonus
  • bonus.getY() - Функция возвращает координату бонуса по оси y.
  • bonus.getX() - Функция возвращает координату бонуса по оси x.
  • bonus.getType() - Функция возвращает тип бонуса(всего 3 типа бонусов: здоровье, оружие, щиты).

SpaceCraft
  • spaceCraft.weapon - Обращение к оружию, для выполнения действий с оружием.
  • spaceCraft.getHealth() - Функция возвращает уровень здоровья вашего корабля в данный момент.
  • spaceCraft.getShield() - Функция возвращает уровень щитов вашего корабля в данный момент.
  • spaceCraft.getX() - Функция возвращает координату корабля по оси х.
  • spaceCraft.getY() - Функция возвращает координату корабля по оси y.
